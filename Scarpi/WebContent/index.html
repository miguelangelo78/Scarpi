API Syntax
  • Normal call: [{"url":"www.yourwebsite.com", "targets": ["css_selector1", "img"], "unescape": false}]
  • Render HTML: [{"url":"www.yourwebsite.com", "targets": ["css_selector1", "img"], "unescape": true}]
  • Get HTML attribute: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | text", "img | src"], "unescape": false}]
  • Get Nth element: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | text | 10", "img | src | 5"], "unescape": false}]
  • Get Nth element with length: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | text | 10 | 5", "img | src | 5 | 2"], "unescape": false}]
  • Get Nth element with length without specifying attribute: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | 10 | 5", "img | 5 | 2"], "unescape": false}]
  • Standard HTML Attributes supported:
    • Get HTML - HTML
    • Plain text - TEXT
    • HREF property - HREF
    • Source - SRC
    • Element tag - TAG
    • Inner HTML - INNER
    • All attributes - _ALL_
    • Custom - "YOUR ATTRIBUTE HERE"
    All attributes are case insensitive
