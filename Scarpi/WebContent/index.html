About

Scarpi is an API written in PHP with the purpose of serving RESTful requests which return data scraped from any website you desire.

The API including this website have been developed by Miguel Santos

API Syntax
  • Normal call: [{"url":"www.yourwebsite.com", "targets": ["css_selector1", "img"], "render": false}]
  • Render HTML: [{"url":"www.yourwebsite.com", "targets": ["css_selector1", "img"], "render": true}]
  • Get HTML attribute: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | text", "img | src"], "render": false}]
  • Get Nth element: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | text | 10", "img | src | 5"], "render": false}]
  • Get Nth element with length: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | text | 10 | 5", "img | src | 5 | 2"], "render": false}]
  • Get Nth element with length without specifying attribute: [{"url":"www.yourwebsite.com", "targets": ["css_selector1 | 10 | 5", "img | 5 | 2"], "render": false}]
  • Minimalistic (the API tries to guess): [{"url":"www.yourwebsite.com"}]
  • Standard HTML Attributes supported:
    • Get HTML - HTML
    • Plain text - TEXT
    • HREF property - HREF
    • Source - SRC
    • Element tag - TAG
    • Inner HTML - INNER
    • All attributes - _ALL_
    • Custom - "YOUR ATTRIBUTE HERE"
    All attributes are case insensitive
